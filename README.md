# Market Basket Analysis

## Project Overview
This project implements a Market Basket Analysis system to discover purchasing patterns and relationships between products using transactional data. The goal is to extract frequent itemsets and association rules that can support business decision-making such as product bundling and promotions.

## Technologies Used
- Python
- PySpark (FP-Growth algorithm)
- Streamlit (Interactive web application)
- Pandas
- Plotly
- Matplotlib
- NetworkX

## Features
- Data preprocessing and cleaning
- Frequent itemsets generation using FP-Growth
- Association rules analysis (support, confidence, lift)
- Interactive dashboards and visualizations
- Business insights and recommendations
- Export results and cleaned data
